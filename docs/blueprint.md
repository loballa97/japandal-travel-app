# **App Name**: GoEasy

## Core Features:

- Location Input: Allow users to input their starting location and destination.
- Route Display: Display available routes with estimated time of arrival and cost.
- Mode Selection: Allow users to select their preferred mode of transportation (e.g., bus, train, car).
- Intelligent Updates: Real-time updates powered by LLM tool based on current traffic and transit conditions.
- Live Tracking: Display user's current location on the map.

## Style Guidelines:

- Primary color: Bright sky blue (#4ECDC4) to convey a sense of freedom and reliability.
- Background color: Very light desaturated blue (#F0F8FF).
- Accent color: Light greenish yellow (#B6FF00) for interactive elements and highlights, to communicate action and progress.
- Font pairing: 'Poppins' (sans-serif) for headlines, 'PT Sans' (sans-serif) for body.
- Use clear and recognizable icons for transportation modes and map markers.
- Clean and intuitive layout with a prominent map view.
- Smooth transitions for map updates and route changes.
